# Hello Streamlit - TODO List

Track your progress as you build your first Streamlit app!

## Steps

- [ ] **Step 1**: Create a basic title
  - Import streamlit
  - Add your first title using `st.title()`

- [ ] **Step 2**: Add page configuration
  - Set page title, icon, and layout
  - Use `st.set_page_config()` at the very beginning

- [ ] **Step 3**: Add headers and basic text
  - Create header and subheader
  - Display regular text with `st.write()`

- [ ] **Step 4**: Add markdown formatting
  - Use `st.markdown()` for rich text
  - Include bold, italic, lists, and links

- [ ] **Step 5**: Display code snippets
  - Show Python code examples
  - Use `st.code()` with syntax highlighting

- [ ] **Step 6**: Add status messages and dividers
  - Display info and success messages
  - Add dividers and captions for visual separation

## Learning Goals

By completing this project, you will learn:
- Basic Streamlit imports and setup
- Text display methods (title, header, write, markdown)
- Code snippet display
- Status messages and UI organization
- Page configuration options
