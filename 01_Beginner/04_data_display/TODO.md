# Data Display - TODO List

Learn different ways to display and interact with data in Streamlit!

## Steps

- [ ] **Step 1**: Create sample data
  - Build a DataFrame from a dictionary
  - Display basic data with `st.write()`

- [ ] **Step 2**: Calculate and display key metrics
  - Extract statistics from DataFrame
  - Use `st.metric()` in columns

- [ ] **Step 3**: Compare display methods
  - Use `st.dataframe()` for interactive tables
  - Use `st.table()` for static display

- [ ] **Step 4**: Add column selection
  - Implement `st.multiselect()` for filtering
  - Display selected columns dynamically

- [ ] **Step 5**: Add JSON display option
  - Use `st.checkbox()` for toggle
  - Display data in JSON format with `st.json()`

- [ ] **Step 6**: Add statistics and editing
  - Show DataFrame statistics with `describe()`
  - Enable data editing with `st.data_editor()`

## Learning Goals

By completing this project, you will learn:
- Creating pandas DataFrames from dictionaries
- Calculating statistics from DataFrames
- Different data display methods in Streamlit
- Interactive vs static table displays
- Column filtering with multiselect
- JSON data representation
- DataFrame statistics and descriptions
- Interactive data editing
