# Image Gallery - TODO List

Create an image gallery with upload, display, and metadata features!

## Steps

- [ ] **Step 1**: Set up file uploader
  - Create file uploader widget
  - Configure for multiple image files
  - Support PNG, JPG, JPEG formats

- [ ] **Step 2**: Display a single image
  - Import PIL for image handling
  - Open and display the first uploaded image

- [ ] **Step 3**: Display all images in a list
  - Loop through uploaded files
  - Display each image with caption

- [ ] **Step 4**: Create grid layout
  - Organize images in a 3-column grid
  - Calculate rows needed dynamically

- [ ] **Step 5**: Add image information
  - Extract file size and dimensions
  - Display metadata below each image

- [ ] **Step 6**: Add expandable info sections
  - Use expanders for image details
  - Add instructions when no images uploaded

## Learning Goals

By completing this project, you will learn:
- File uploading with Streamlit
- Working with PIL (Python Imaging Library)
- Displaying images with captions
- Creating grid layouts with columns
- Extracting image metadata
- Loop and conditional logic
- Expandable UI sections
