# Pflichtenheft: Stock Price Tracker

## Expected Functionality
Display stock price data and charts (demo data).

## Input
- Stock symbol selection

## Expected Output
- Price chart
- Key metrics
- Historical data

## Dependencies
```txt
streamlit>=1.28.0
plotly>=5.17.0
pandas>=1.5.0
```

## Usage
```bash
streamlit run app.py
```
