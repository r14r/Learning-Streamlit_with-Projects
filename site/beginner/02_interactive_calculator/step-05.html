02 Interactive Calculator — Step 5

Level: Beginner Projekt: 02 Interactive Calculator
Zurueck zu Step 4: Add operation selection Weiter zu Step 6: Handle division by zero

Was zu tun ist

Implement calculation logic

  • Add calculate button
  • Perform operations based on selection
  • Display results with formatting

Tipps & Code

Key Concept: Buttons trigger actions when clicked; use if/elif for different operations.


if st.button("Calculate", type="primary"):
    result = None

    if operation == "Addition (+)":
        result = num1 + num2
    elif operation == "Subtraction (-)":
        result = num1 - num2
    elif operation == "Multiplication (×)":
        result = num1 * num2
    elif operation == "Division (÷)":
        result = num1 / num2

    if result is not None:
        st.success(f"Result: {result:.2f}")

Tip: type="primary" makes the button stand out with accent color.
